# Server creation

We have over **24** different server types you can choose from to start your hosting experience!

<Tip>
  For the purpose of this page we will just be showing the create server list,
  you can look at the other guides to see how to create and work with them
</Tip>

(For non-donators)
By running `DBH!server create list` you can see the list of all the servers you can create!

![Server Creation List (For Non Donator Members)](/content/create-server/normal-server-create-list.png)

(For donators)
By running `DBH!server create-donator list` you can see the list of all the servers you can create!

![Server Creation List (For Non Donator Members)](/content/create-server/donator-server-create-list.png)
